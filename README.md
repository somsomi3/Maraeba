    1. WebRTC 개념 공부

    WebRTC란 웹 기반 실시간 음성, 영상 통신기술

    Peer-to-Peer 기술을 사용하여, 서버를 거치지 않고 브라우저 간에 직접적인 연결을 가능하게 함

    WebRTC의 구성 요소는 미디어 스트림, 시그널링, NAT 트래버설

    WebRTC의 종류는 Mesh, SFU, MCU

- WebRTC서버 구축을 위해 필요한 4가지
  1. 시그널링 서버
  2. Stun 서버
  3. Turn서버
  4. 미디어 서버
